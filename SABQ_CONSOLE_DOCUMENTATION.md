# توثيق لوحة سبق الذكية - Sabq Smart Console Documentation

## 6. نظام القوالب - Templates System

### نظرة عامة
نظام إدارة القوالب يسمح بالتحكم الكامل في العناصر الثابتة والتصميمية للموقع بطريقة ذكية وديناميكية.

### أنواع القوالب المدعومة

#### 1. قوالب الهيدر (Header Templates)
- **الشعار**: رفع وتحديد أبعاد الشعار
- **القوائم**: إضافة وترتيب روابط التنقل
- **الشريط العلوي**: الأخبار العاجلة، التاريخ، الطقس
- **روابط التواصل**: أيقونات وسائل التواصل الاجتماعي
- **المظهر**: الألوان والثيمات المخصصة

#### 2. قوالب الفوتر (Footer Templates)
- **أقسام الفوتر**: روابط منظمة في أقسام
- **النشرة البريدية**: نموذج الاشتراك
- **تطبيقات الجوال**: روابط iOS و Android
- **حقوق النشر**: نص قابل للتخصيص

### ميزات النظام

#### التحكم الذكي
1. **جدولة القوالب**: تفعيل تلقائي حسب التاريخ والوقت
2. **الاستهداف الجغرافي**: قوالب مخصصة حسب الدولة
3. **ربط بالتصنيفات**: قوالب مختلفة لكل قسم
4. **القوالب الافتراضية**: قالب أساسي لكل نوع

#### واجهة المستخدم
- محرر مرئي سهل الاستخدام
- معاينة مباشرة للتغييرات
- إدارة متعددة للقوالب
- نظام تفعيل/تعطيل سريع

### الاستخدام

#### إنشاء قالب جديد
1. الذهاب إلى القوالب من القائمة الجانبية
2. اختيار نوع القالب (هيدر/فوتر)
3. النقر على "قالب جديد"
4. تعبئة البيانات والتصميم
5. حفظ وتفعيل القالب

#### تحرير قالب موجود
1. النقر على أيقونة التعديل
2. التنقل بين التبويبات (عام/تصميم/شروط)
3. إجراء التعديلات المطلوبة
4. حفظ التغييرات

### قاعدة البيانات

```sql
CREATE TABLE templates (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  type TEXT CHECK (type IN ('header', 'footer', 'sidebar', 'banner')),
  content JSONB NOT NULL,
  is_active BOOLEAN DEFAULT FALSE,
  is_default BOOLEAN DEFAULT FALSE,
  starts_at TIMESTAMP,
  ends_at TIMESTAMP,
  country_code TEXT,
  category_id INT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### API Endpoints

```
GET    /api/templates          - جلب القوالب
POST   /api/templates          - إنشاء قالب
GET    /api/templates/:id      - جلب قالب محدد
PATCH  /api/templates/:id      - تحديث قالب
DELETE /api/templates/:id      - حذف قالب
POST   /api/templates/:id/set-default - تعيين كافتراضي
```

### أمثلة للاستخدام

#### قالب رمضان
- تفعيل تلقائي من 1-30 رمضان
- شعار وألوان خاصة
- رسائل ترحيب مخصصة

#### قالب اليوم الوطني
- مخصص للسعودية فقط
- ألوان العلم السعودي
- شعارات وطنية

### ملاحظات للمطورين
- القوالب تُحفظ بصيغة JSON في قاعدة البيانات
- يمكن توسيع النظام لدعم أنواع قوالب إضافية
- النظام يدعم المتغيرات الديناميكية مثل {year} للسنة الحالية
