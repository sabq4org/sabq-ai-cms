## استخدام الألوان في Tailwind

### Primary Colors
- `text-primary` → `text-blue-600`
- `bg-primary` → `bg-blue-600`
- `border-primary` → `border-blue-600`
- `hover:bg-primary` → `hover:bg-blue-700`

### Status Colors
- `text-success` → `text-green-600`
- `bg-success` → `bg-green-100`
- `text-warning` → `text-yellow-600`
- `bg-warning` → `bg-yellow-100`
- `text-danger` → `text-red-600`
- `bg-danger` → `bg-red-100`

### Gray Scale
- `text-gray-900` → الأسود الأساسي
- `text-gray-700` → نص ثانوي
- `text-gray-500` → نص مخفف
- `bg-gray-50` → خلفية فاتحة جداً
- `bg-gray-100` → خلفية فاتحة
- `border-gray-200` → حدود خفيفة
