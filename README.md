## 🌟 نبذة عن المشروع

نظام إدارة محتوى متطور لصحيفة "سبق" الإلكترونية، مدعوم بأحدث تقنيات الذكاء الاصطناعي لتحسين تجربة القراء والمحررين على حد سواء. يتميز النظام بواجهة عصرية وأداء عالي مع قدرات تحليلية متقدمة.

### 🎨 نظام التصميم الأزرق
تم تطوير نظام تصميم مخصص بدرجات الأزرق الخفيف يعكس هوية صحيفة سبق الاحترافية. يمكنك مشاهدة جميع المكونات في صفحة `/components-showcase`.

- **Next.js 14** - إطار عمل React متقدم
- **TypeScript** - للأمان في الكتابة
- **TailwindCSS** - للتصميم السريع والمرن

### قاعدة البيانات والتخزين
- **PostgreSQL** - قاعدة البيانات الرئيسية

### الذكاء الاصطناعي والتحليلات

- **Custom Fine-tuned Models** - نماذج مخصصة للعربية

## 🚦 البدء السريع

### المتطلبات الأساسية
- Node.js 18+ و npm 9+
- PostgreSQL 14+
- Redis 7+
- حساب OpenAI API

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/sabq/sabq-ai-cms.git
cd sabq-ai-cms
```

2. **تثبيت المكتبات**
```bash
npm run install:all
```

3. **إعداد متغيرات البيئة**
```bash
cp .env.example .env
# قم بتعديل ملف .env وإضافة المفاتيح المطلوبة
```

4. **إعداد قاعدة البيانات**
```bash
cd backend
npx prisma migrate dev
npx prisma db seed
```

5. **تشغيل المشروع في وضع التطوير**
```bash
npm run dev
```

سيعمل التطبيق على:
- Frontend: http://localhost:3000
- Backend API: http://localhost:4000
- GraphQL Playground: http://localhost:4000/graphql

## 📝 الاستخدام

### للمحررين
1. الدخول إلى لوحة التحكم من `/admin`
2. إنشاء مقال جديد بالضغط على "مقال جديد"
3. استخدام المحرر الذكي لكتابة المحتوى
4. الاستفادة من اقتراحات الذكاء الاصطناعي
5. جدولة النشر أو النشر الفوري

### للمطورين
```typescript
// مثال: استخدام API لجلب المقالات
import { ArticleService } from '@/services/article.service';

const articles = await ArticleService.getLatest({
  limit: 10,
  category: 'سياسة'
});
```
